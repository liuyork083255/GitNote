
1		在分区表里面，如果刚好用了一个索引字段做为分区划分，那么在查询的时候，如果条件范围跨过了一个分区，那么这个时候，这个索引会失效
	比如 date_time 字段是个索引，而且用month函数 按月分区，如果一个sql 里面的where 范围跨过了两个月，那么这个索引会失效。
	
	
	
	
2		如果一个sql 语句 用explain 分析出来，发现可以使用多个 索引，那么mysql会默认使用一个，但是具体是哪一个mysql自己选，
	注意：mysql默认选的并不一定会是最优的，这个时候可以自己选择使用哪个索引。
	
	语法：select * from tableA user index (index_name) where ...
		

		
3		如果where 中的一个字段有索引，那么这个索引不能参与任何操作，比如 将 datetime 转为 date(datetime)，这时候就丧失了索引
	又例如 id 是一个int类型的索引，如果id/100 = 5 ，也会丧失索引值，但是 id = 5*100; 就不会丧失索引。
